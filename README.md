# 🐾 PataAmiga — Sistema de Adoção de Cachorros

Projeto acadêmico desenvolvido para a disciplina de **Computação em Nuvem**, com o objetivo de demonstrar o deploy de uma aplicação web estática na plataforma **Netlify**.

## 📋 Sobre o Projeto

A **PataAmiga** é uma landing page demonstrativa de um sistema de adoção de animais. A proposta é conectar pessoas que desejam adotar um cachorro com tutores que, por algum motivo, precisam encontrar um novo lar para seu pet — tudo de forma simples, gratuita e acessível.

### Funcionalidades demonstradas:
- Listagem de pets disponíveis para adoção
- Formulário de solicitação de adoção
- Formulário para cadastro de pet para doação
- Design responsivo (mobile e desktop)

## 🌐 Tecnologias Utilizadas

- **HTML5 / CSS3 / JavaScript** — aplicação 100% estática, sem dependências externas
- **Netlify** — plataforma de hospedagem e deploy contínuo na nuvem
- **Google Fonts** — tipografia (Fraunces + DM Sans)
- **Unsplash** — imagens via CDN público

## 🚀 Como executar localmente

Basta abrir o arquivo `index.html` diretamente no navegador — nenhuma instalação necessária.

## ☁️ Deploy

Aplicação publicada via **Netlify** com deploy por arrastar e soltar (*drag and drop*), demonstrando a facilidade de publicação de aplicações estáticas em ambientes de nuvem.
